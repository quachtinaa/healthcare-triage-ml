# healthcare-triage-ml

## Abstract
Accurate triage in emergency rooms is very crucial for efficient patient care and resource allocation. This project explores the use of machine learning, in particular neural networks, to predict patient triage levels based on key clinical indicators and patient information such as age and pain levels. 

This project was done using a synthetic mdical triage dataset from Kaggle. Using this dataset, the model performs data analysis to understand the relationships between patient characteristics and triage level, our target variable. 

A neural network model was implemented using PyTorch to classify patients into different triage levels. Additionally, an interactive dashboard was developed to demonstrate how the trained model can be used to predict triage levels for new patient inputs.

The results show how machine learning models can effectively assist the healthcare industry and its professionals in prioritizing patients more effictively, and highlight the potential of data-driven decision support systems in the field.
